# Rize Skill Training — CTA System V1

This file sets the unified CTA hierarchy for the website.

## Unified hierarchy

1. Start Here.
2. Explore Community.
3. Explore Career.
4. Explore Services.
5. Talk About a Partnership.
6. Book a Discovery Call.

## Usage rules

- Use Start Here for the homepage hero and first-touch navigation.
- Use Explore Community, Explore Career, and Explore Services for broad routing.
- Use Talk About a Partnership for community organizations, nonprofits, libraries, employment partners, and other collaboration-focused visitors.
- Use Book a Discovery Call for higher-intent service and implementation conversations.
- Use Request This Workshop and Talk to Us About a Course only inside the Learning page where the action is specific.

## Page mapping

### Home
Primary: Start Here.
Secondary: Explore Community, Explore Career, Explore Services, Talk About a Partnership.

### Learning
Primary browsing: Explore Community, Explore Career, Explore Creative.
Specific actions: Request This Workshop, Talk to Us About a Course.
Fallback: Get in Touch.

### Services
Primary: Book a Discovery Call.
Secondary: Talk About a Partnership.

### Partnerships
Primary: Talk About a Partnership.
Secondary: Book a Discovery Call.

### About
Primary: Book a Discovery Call.
Secondary: See Our Programs, Explore Funded Partnerships.

## Tone notes

- Keep CTA language direct and active.
- Use the fewest unique CTAs possible.
- Repeat the same CTA on a page rather than inventing new ones.
- Prefer action words that match the page purpose.
